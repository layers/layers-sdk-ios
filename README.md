# Layers SDK for iOS

## Installation

### Swift Package Manager

Add to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/layers/layers-sdk-ios", from: "1.0.0")
]
```

Or in Xcode: File → Add Package Dependencies → enter `https://github.com/layers/layers-sdk-ios`

## Documentation

See [docs.layers.com](https://docs.layers.com) for integration guides.
